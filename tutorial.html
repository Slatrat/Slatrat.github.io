How to Use

Follow these steps to generate in-game text for DayZ:

  1. Enter the item file name (e.g., WoodenPlank).
  2. Specify the width and height of the item in inches.
  3. Type the text you want to display in the game.
  4. Select the font style for your text from the dropdown menu.
  5. Set the letter height in feet.
  6. Optionally, provide the starting coordinates (X, Y, Z) in meters where you want the text to appear.
  7. Optionally, enter the orientation in degrees to set the direction of the text.
  8. Click the Generate Code button to create the code.
  9. Copy the generated code from the text area and integrate it into your DayZ mod scripts.

Once integrated, the items will be placed in the game world forming the text at the specified location and orientation.
